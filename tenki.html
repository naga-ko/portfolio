作品名 天気文字数対決
ジャンル Webアプリ制作
制作期間 約1ヶ月　　2023年5月〜
制作人数 1人
使用技術、ツール HTML / CSS / JavaScript / Sass / Jquery / VisualStudioCode
URL https://koki.main.jp/tenki/

概要

JavaScript演習の授業で天気予報APIを使ったWebアプリ制作をしました。

Point1　　対決

天気を取得して画像を変える...みたいなやつだと面白くないと思いました。そしてjsonのデータを見たときに天候が英語で書いてある物があったのでこれを利用して都道府県の天気を取得し、その文字数で対決するアプリを作りました。

Point2　　天気の取得

今回私は授業資料に乗っていた３つの都道府県だけでなく全部の４７都道府県の天気の取得をしました。 どこに何が書かれているのかを把握することに時間がかかりましたがなんとかできました。

Point3　　デザイン

対決するといったら自分はトランプが一番最初に思い浮かんだので背景や天気のマークなどをトランプのようにしてデザインしました。

作品制作を通して

APIを使ってみたら意外と面白いものができそうだなと感じました。 今後APIを使うことがあるかはわからないけどこうやるんだなって感覚はつかめました。 コードの意味がわかって自分の頭の中で思っていることができたときは楽しかった。
